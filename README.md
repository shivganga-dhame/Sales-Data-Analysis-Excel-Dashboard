# 📊 Sales Data Analysis Dashboard - Excel Project

## 🔍 Project Overview
This project analyzes sales data and builds an interactive dashboard using Microsoft Excel. The objective is to generate business insights such as profit analysis, order trends, customer segmentation, and payment mode performance.

---

## 🛠 Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Charts (Pie, Line, Bar)
- Conditional Formatting
- KPI Cards
- Data Cleaning
- Top/Bottom Analysis
- Unique Customer Extraction

---

## 📈 Key Analysis Performed

### ✅ Top & Bottom 10 Products by Profit
Identified highest and lowest performing products.

### ✅ Above Average Sales Analysis
Compared product performance against average sales.

### ✅ Payment Mode Wise Orders
Analyzed order distribution across COD, Cards, and Online.

### ✅ Month Wise Sales Trend
Visualized sales growth across months.

### ✅ KPI Metrics
- Total Profit
- Total Orders
